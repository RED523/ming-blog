------

## PauseCat 是什么？

**PauseCat** 是一个基于 Chrome Manifest V3 的扩展。
它的目标只有一个：**在你快失控的时候，及时踩一脚刹车**。

核心逻辑非常轻量：

1. 你设置专注时长（默认 60 分钟）
2. 到点后触发休息提醒（默认 5 分钟）
3. 通过全屏猫咪动画/视频中断当前分心流

不是鸡血，不是自律口号，是一个真正会“出手干预”的工具。

## 为什么它比普通计时器更有效？

普通计时器常见问题：

- 提醒太弱，点掉继续刷
- 后台计时不准确
- 对真实浏览场景不敏感

PauseCat 的设计重点是“**真实使用场景下的有效提醒**”：

- 只统计**当前聚焦窗口 + 活动标签页**的目标网站时长
- 系统空闲/锁屏、后台标签页不计时
- 支持默认站点和自定义域名（含子域名匹配）
- 到点后是全屏级提醒，不容易无视

------

## 功能亮点一览

- 可配置浏览时长与休息时长
- 预设分心网站：X/Twitter、YouTube、Reddit、bilibili、知乎、抖音
- 支持自定义域名匹配（如 `example.com` 自动覆盖 `www.example.com`）
- 猫咪视频素材可替换（也有动画兜底）
- 轻量、本地运行、开箱可用

## 3 分钟上手安装（本地）

```bash
git clone https://github.com/RED523/PauseCat.git
```

然后：

1. 打开 Chrome，访问 `chrome://extensions`
2. 开启右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择 `PauseCat` 项目文件夹
5. 点击浏览器工具栏中的 PauseCat 图标，开始配置
